## Project Context

- **Project:** Titan-Takeaway
- **Owner:** Darren Pratt
- **Created:** 2026-03-25
- **Goal:** Build a training application and documentation to teach developers OpenTelemetry, Prometheus, Grafana, Loki, Elasticsearch, and Jaeger using a microservice "Titan Takeaway" system.
- **Architecture:** Dockerized microservices with SQL Server backing services.
- **Training Scenarios:** payment 503 failures, kitchen latency under load, stuck preparing orders, retry storms.
